#  Multi-Agent AI Research System

An enterprise-style multi-agent orchestration system for conducting automated investigative research using large language models and external data sources.

##  Features

- Multi-agent architecture (Planner, Researcher, Analyst, Writer, Fact Checker)
- Parallel web research using DuckDuckGo and news extraction
- Source deduplication and credibility scoring
- Structured claim extraction and verification
- Iterative orchestration with retry and failure handling
- Config-driven research depth
- CLI interface with live progress visualization
- Markdown report generation
- Designed to handle real-world API limits gracefully
